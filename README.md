# Job Posting Portal

## Project Overview
The Job Posting Portal is a web-based application that connects employers and job seekers on a single platform. It allows companies to post job openings and enables candidates to search for jobs based on their interests and skills. The platform simplifies the recruitment process by organizing job listings in one place.

## Features
- User Registration and Login
- Job Posting by Companies
- Job Search Functionality
- Job Listings Display
- Job Application System
- User-Friendly Interface
- Centralized Job Information

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js

## Installation & Setup

1. Clone the repository:

  git clone https://github.com/your-username/job-posting-portal.git

2. Navigate to the project folder:

  cd job-posting-portal

3. Install dependencies:

  npm install

4. Start the server:

  node server.js

5. Open in browser:

  http://localhost:3000
